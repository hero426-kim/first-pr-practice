# first-pr-practice

A tiny practice project for learning the GitHub pull request workflow.

## What's here

A small `calculator` module with basic arithmetic operations and a test suite,
used as a sandbox for practicing branches, commits, and pull requests.

## Usage

```bash
python -m pytest
```

## Contributing

This repo is for practicing the PR workflow: create a branch, make a small
change, and open a pull request agains `main`.
